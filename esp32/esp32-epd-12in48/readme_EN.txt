This file is to help you use this routine.

1. Basic information:
This routine was developed based on Arduino, and the routines were verified on the E-Paper ESP32 Driver Board.

2. Basic use:
     Copy the entire EPD folder to the \hardware\espressif\esp32\libraries folder in the Arduino installation path.
     Then open the Arduino IDE, find the EPD12in48 in the file =》example, open the corresponding demo, and download it to the E-Paper ESP32 Driver Board.
